# Planen för vidarebefordran av händelser

Formboken för vidarebefordran av händelser visar hur data som samlats in med Adobe [!DNL Experience Platform] Web och Mobile SDK kan vidarebefordras från [!DNL Experience Platform] [!DNL Edge Network] till önskat mål. Du kan vidarebefordra alla rådata som samlats in från SDK:n eller specifika data baserade på händelser och regler som konfigurerats i taggegenskaperna (tidigare Launch).

## Användningsfall

* Samla in data från webben eller mobiler med en enda samlingstagg, vilket gör kodvikten lättad i webbläsare och appar. Sprid insamlade data till olika slutpunkter för en enda datakälla för datainsamling.
* Vidarebefordra insamlade data till partnerapplikationer eller lagringsplatser för data för att skapa insikter och applikationer mot insamlade data.

## Tillämpningar

* Datainsamling för Adobe [!DNL Experience Platform]
