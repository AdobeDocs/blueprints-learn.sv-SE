# Öka webbplatsengagemanget

Förbättra tiden på webbplatsen, sidor per session och interaktion med webbinnehåll genom relevanta upplevelser. Detta mål är inriktat på att leverera kontextuella, personaliserade webbupplevelser som håller besökarna engagerade och går igenom innehåll och konverteringsvägar.

## Nyckeltal för prestanda

Följande nyckeltal mäter framstegen mot detta mål.

| KPI | Beskrivning |
| --- | --- |
| Tid på sidan | Genomsnittlig tid som besökare tillbringar på nyckelsidor |
| Engagemang | Sammansatt poäng för interaktioner som klickningar, rullningar och innehållskonsumtion |
| Konverteringsgrader | Procent engagerade besökare som slutfört önskade åtgärder |

## Relaterade användningsfallsmönster

Följande användningsmönster stöder detta affärsmål.

| Mönster | Kategori | Hur det stöder detta mål |
| --- | --- | --- |
| [Anonym besökare på Personalization](/help/blueprints/use-case-patterns/personalization/anonymous-visitor-web-personalization.md) | Personalisering | Levererar personaliserat webbinnehåll till oidentifierade besökare baserat på sessionsbeteenden för att öka engagemanget |
| [Known-Visitor Web/App Personalization](/help/blueprints/use-case-patterns/personalization/known-visitor-web-app-personalization.md) | Personalisering | Personaliserar webb- och appupplevelser för autentiserade besökare med fullständiga profildata |
| [Beteenderekommendation](/help/blueprints/use-case-patterns/personalization/behavioral-recommendation.md) | Personalisering | Levererar AI-styrt innehåll och produktrekommendationer för att hålla besökarna engagerade |
| [Brand Concierge Conversational Experience](/help/blueprints/use-case-patterns/conversational-experience/brand-concierge-conversational-experience.md) | Konversationsupplevelser | Tillhandahåller artificiell intelligens (AI) som hjälper besökare att hitta innehåll |
