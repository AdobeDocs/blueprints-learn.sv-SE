# Förbättra datakvalitet och styrning

Säkerställ rena, fullständiga och kompatibla data för korrekt målinriktning, minskat avfall och tillförlitlig analys. Detta mål är inriktat på att skapa den grund för data som krävs för all marknadsföring i efterföljande led, vilket säkerställer att kunddata är korrekta, konsekventa, välkontrollerade och uppfyller alla regler.

## Nyckeltal för prestanda

Följande nyckeltal mäter framstegen mot detta mål.

| KPI | Beskrivning |
| --- | --- |
| Effektivitet | Förbättrad databehandling och exakthet i användningen |
| Kostnadsbesparingar | Minskat avfall från målinriktning baserat på felaktiga eller duplicerade data |
